Javascript Object

Developers create their own objects by creating instances of the Object type and adding properties and/or methods to it.

New objects are created by using the new operator followed by a constructor. A constructor is simply a function whose purpose is to create a new object.

var person = new Object();

Process:

  • - Object() constructor creates a simple object with only the default properties and methods
  • - creates a new instance of the Object refference type
  • - stores the instance in the variable person

Create an instance of an Object

There are two ways to explicitly create an instance of Object:

new operator

var person = new Object();
person.name = "Nicholas";
person.age = 29;

object literal notation

var person = {
  name: "Nicholas",
  age: 29
}
